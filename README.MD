# Update Projects
Ferramento para atualizar todos os clientes web e android.
##

# Configuração do .env
```
cp .env-exemplo .env
```
No arquivo .env, lembre de alterar os valores das variáveis.

# Requisitos
Node >= 16 

# Instalação dos módulos
```
  npm install 
```

# Atualizar um projeto específico

```
python3 update.py servicos 
```
